Verify [% server.ip %]

To verify that you operate the server retrieve a verification token by doing an HTTP request from the NTP server to [% validation_server | html %]. The validation server will return a URL for you to load to finish the validation process.

For example one of these commands:

curl --interface [% server.ip %] [% validation_server | html %]

wget --bind-address=[% server.ip %] -O- [% validation_server | html %]

The server also responds to non-TLS requests.
